Pour aller plus loin

Construire un starter pedagogique

Listez les starters disponibles puis construisez-en un dans ce projet. 

forge starter:list
forge starter:build welcome

Generer une entite

Creez le model, le validator, le controller, les vues et les routes d'une entite. 

forge make:entity NomEntite

Preparer la base de donnees

Configurez MariaDB dans env/dev puis initialisez la base. 

forge db:init
